## RAG 기반 질의응답 프로젝트

### 데이터 품질 검증 및 RAG 입력 데이터 정합성 개선

비정형 공고 문서 데이터를 기반으로 RAG 질의응답 시스템을 구축하기 위한
**데이터 품질 검증과 입력 데이터 정합성 개선**을 담당했습니다.

**Role**

* 데이터 엔지니어링
* 데이터 정합성 검증
* 수동 보정 및 예외 처리 규칙 정리
* 프롬프트 구조화
* 팀 공유용 문서화

**What I Did**

* 비정형 공고 문서 데이터 수동 보정 전수조사
* 메타데이터와 원문 문서 내용이 불일치하는 데이터 오염 케이스 발견
* RAG 환각 방지를 위한 데이터 필터링 기준 정리
* 문서마다 다르게 표현된 금액·규모 관련 용어를 표준 항목 기준으로 통합
* Search, Extraction, Output 3단계 모듈형 프롬프트 구조 설계
* 수동 보정 과정에서 발견한 예외 케이스와 보정 로직 문서화
* 데이터 정합성 확보를 위한 예외 처리 규칙 정리

**What I Learned**

RAG 서비스 품질은 모델 성능뿐 아니라
입력 데이터의 정합성, 용어 통일, 예외 처리 규칙에 크게 영향을 받는다는 점을 경험했습니다.
