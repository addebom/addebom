## About Me

사회복지와 신학을 전공하며 사람과 조직을 이해하는 관점을 배웠습니다.

이후 웹디자인과 온라인 쇼핑몰 운영 업무를 경험했고, 독학으로 영상 편집을 공부하여 게임 콘텐츠 제작 조직에서 영상 팀장(PD) 역할을 수행했습니다.

현재는 AI 프로젝트를 통해 데이터 품질 검증, 서비스 흐름 분석, 프로젝트 운영 경험을 쌓고 있습니다.

제가 관심 있는 것은 기술 자체보다 문제를 발견하고, 정보를 이해하기 쉬운 구조로 정리한 뒤, 실제 서비스로 연결하는 과정입니다.
