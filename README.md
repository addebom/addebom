이후 온라인 쇼핑몰 운영과 웹디자인 업무를 경험했고, 독학으로 영상 편집을 공부하여 게임 콘텐츠 제작 조직에서 영상 팀장(PD) 역할을 수행했습니다.

현재는 AI 엔지니어 부트캠프를 통해 데이터 품질 검증, AI 서비스 흐름 분석, 프로젝트 운영 경험을 쌓고 있습니다.

제가 관심 있는 것은 기술 자체보다 사용자의 문제를 발견하고, 이해하기 쉬운 구조로 정리한 뒤, 실제 서비스로 해결하는 과정입니다.
